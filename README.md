# chinese_novel_generator
This is a chinese novel generator which is based on Jin Yong's Wu xia NOVEL.



这是一个可以学习和生成中文小说的生成器，用神经网络来训练的。
主要是基于金庸的武侠小说来完成的。

需要Tensorflow等依赖

运行main.py即可
